### 🏢 Nexus Portal — Enterprise Management Platform
A cross-platform internal management portal built for a 
multi-subsidiary organisation. Handles financial requisitions, 
HR leave management, vehicle tracking, lease management, 
and policy/claims — all from a single React Native codebase 
running on Android, iOS, and Web.

**Tech Stack:** TypeScript · React Native · Expo v54 · Expo Router · 
React 19 · Role-Based Access Control · Cross-Platform

> 🔒 *Sanitised for portfolio use — original built for enterprise client*
